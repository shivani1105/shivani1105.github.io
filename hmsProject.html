The Problem: Fragmented Healthcare Operations

Hospitals often rely on manual processes and disconnected systems to manage appointments, patients, staff, billing, and lab results. This results in delayed service, inconsistent data, and a poor patient experience. We needed a centralized, secure system to streamline these operations.

The Solution: A Modular Hospital Management System

Homepage showcasing modules like Appointments, Lab, Pharmacy, and Doctors

Microservices architecture with API Gateway, JWT Auth, and independent service databases

I developed a full-stack HMS using Angular for the frontend and ASP.NET Core Web API for backend services. Key services like Patient, Appointment, Authentication, and API Gateway were built using microservices with JWT-based role authentication.
